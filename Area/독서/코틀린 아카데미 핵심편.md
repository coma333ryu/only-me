## 3장 변수

## 변수 선언
* variable(변수) var
	* 읽고 쓸 수 있는 변수. 초기화 후에도 언제든 값 변경 가능한 변수
* value(값) val
	* 읽기만 가능한 변수. 초기화 후 변경 불가능.

## 변수 타입 명시
* 변수 이름 다음에 콜론(:)을 붙여 타입 명시
```
var a: Int = 10
```

## 리터널(?)



* 코틀린에서는 모든 값이 객체로 생각되기 때문에 원시 타입이 없음.
* 모든 값은 메서드를 제공함.
* 값의 타입을 제네릭 타입 인수로 사용가능.
* 숫자를 나타내는 기본 타입은 Number 타입의 서브타입

```
+=:a += b     // a = a + b
-=:a -= b     // a = a - b
*=:a *= b     // a = a * b
/=:a /= b     // a = a / b
%=:a %= b     // a = a % b
```
